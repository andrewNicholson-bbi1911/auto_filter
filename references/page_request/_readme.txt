Для правильного добавления сгенерированного кода следуй следующей инструкции:

1. Добавим соответствующий api метод из:
        ./page_api.txt
    в соответствующую странице API источнику, обычно находится в:
        lib/data/source/api/source/страничка/та_самая_api.dart


2. Добавь в:
        lib/data/source/api/api.dart
    согласно инструкции код из
        ./api.txt

3. Добавь в соответствующий репозиторий:
        lib/data/repositories/страничка/тот_самый_repository.dart
    согласно инструкции код из 
        ./page_repository.txt
